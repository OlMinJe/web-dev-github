input readonly 속성

readonly 속성은 입력 필드가 읽기 전용이어야 함을 지정합니다.(변경할 수 없음)



input disabled 속성

readonly 속성은 입력 필드가 비활성화되어야 함을 지정함니다.(사용할 수 없고 클릭할 수 없음)



input size 속성

size 속성은 입력 필드의 너비를 지정한다.



input maxlength 속성

maxlenght 속성은 입력 필드에 허용되는 최대 문자 수를 지정합니다.



input max 및 min 속성

min및 max 속성은 입력 요소의 최소값과 최대값을 지정한다.

input multiple 속성

multiple 속성은 사용자가 입력 필드에 둘 이상의 값을 입력할 수 있도록 지정한다.

input placeholder 속성

placeholder 속성은 입력 필드의 예상 값을 설명하는 짧은 힌트를 지정한다.

input step 속성

input step 속성은 입력 요소의 유효한 숫자 간격을 지정한다.
