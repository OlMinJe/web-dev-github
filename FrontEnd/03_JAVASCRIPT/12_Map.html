Map

new Map()

배열로 map 만들기

    const fruitsArray = [
        ["사과", 500],
        ["바나나", 300],
        ["오렌지", 200]
    ];
    const fruits = new Map(fruitsArray);

배열 생성후 set()로 추가하기

    const fruits2 = new Map();
    fruits2.set("사과", 500);
    fruits2.set("바나나", 300);
    fruits2.set("오렌지", 200);

set() 사용시 key의 중복

Map은 key 값을 기준으로 중복을 허용하지 않는다. 만약 동일한 key 값에 다른 value로 set()를 호출하면 나중에 적용된 value로 설정된다.

    fruits.set("딸기", 100);
    fruits.set("사과", 250);
    fruits.set("수박", 400);

delete()

delete(key)는 Key와 일치하는 요소를 삭제한다.

clear()

clear()는 set의 모든 요소를 삭제한다.

has()

has(key)는 key와 일치하는 요소가 있으면 true, 없으면 false를 반환한다.

entries()

[key, value] 형식 Iterator 객체 반환

    const entry = fruits.entries();

keys()

set 요소중 key들만으로 구성된 iterator 객체를 반환

    const keyIterator = fruits.keys();

sets()

set 요소중 value들만으로 구성된 iterator 객체를 반환

    const keyIterator = fruits.values();
